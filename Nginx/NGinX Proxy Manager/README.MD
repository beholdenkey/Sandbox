# Nginx Proxy Manager

source: <https://nginxproxymanager.com/>

## Quick Setup

To use the associated docker-compose file in this directory you must have the following installed.

- Docker-CE
- Docker-Compose

To bring up the stack run the following command

```bash
docker compose up -d
```

Default Admin User:

- Email: admin@example.com
- Password: changeme

### Disclaimer

If you are going to use this for production or even personal use it is highly recommended that you do not use the default settings in the docker-compose.yml file
